Rules and Regulations

01

Scope of Application

  • These rules apply to the entire Minecraft server, including all game modes and related services
  • By joining the server, all players accept these rules as binding
02

In-Game Conduct

Any behavior that negatively impacts gameplay or other players is prohibited. This includes, but is not limited to:

Disrupting teammates

Unauthorized teaming

Spawn camping

AFK / delaying

Random attacking in role-based modes

Offensive builds

03

Modifications & Tools

The use of software or hardware that provides gameplay advantages is strictly forbidden, even if only passively active. Examples include:

Combat Mods

like KillAura, Aimbot, AutoClicker

Movement Aids

like Fly, Speed, Glide, Fast-Fall

Info Mods

like Damage Indicators, Player Radar

Inventory Mods

like Auto-management, Macros

04

Additional Provisions

The Minecraft EULA is part of these server rules and must be followed.

Using stolen, public, or compromised Minecraft accounts is not allowed.
