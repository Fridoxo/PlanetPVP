Voice Chat

Ihr könnt ganz einfach auf CurseForge nach der Voice Chat Mod suchen und herunterladen.

Alternativ gibt es ein Tutorial, das euch durch die Installation führt.

Mods & Installationen für das Voice Chat Tutorial

  • Fabric Installer - Klickt auf „Download universal jar“
  • Fabric API Mod - Rechts auf Download drücken
  • Java Download - Notwendig für Mods

Chunks speichern

Bobby Mod - Speichert Chunks auf eurem PC für unbegrenzte Sichtweite

Bitte nutzt diese Mod wenn möglich, da sie den Server schont.

Night Vision Shader

So aktiviert ihr den Night Vision Shader:

  1. Shader herunterladen
  2. Minecraft öffnen & Server beitreten
  3. „Esc“ drücken & Optionen -> Resource Pack
  4. „VanillaTweaks_...“ per Drag & Drop in Minecraft ziehen
  5. Auf das Pack klicken, sodass es nach rechts rutscht

Server Neustart

Jeden Tag um 5:00 Uhr wird der Server automatisch neu gestartet.

Nützliche Ingame-Befehle

  • /sit - Auf einen Block setzen
  • /lay - Sich hinlegen
  • /bellyflop - Bauchklatscher
  • /spin - Sich drehen
  • /crawl - Kriechen
  • /tps - Zeigt die aktuellen TPS an
  • /tabtps toggle - Aktiviert eine TPS/MSPT Anzeige
  • /status noob - Gibt den Noob-Status
  • /status-clear - Entfernt den Noob-Status
